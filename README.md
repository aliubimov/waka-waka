RFM95 Pin setup
---

NSS -> J57[6]

SCK -> J57[12]

MOSI -> J57[8]

MISO -> J57[10]

3.3V -> J60[8]

GRND -> J60[14]



LCD Pin Setup
---

CS -> J57[6]

SCK -> J57[12]

SDI(MOSI) -> J57[8]

SDO(MISO) -> J57[10]

VCC(5V) -> J60[10]

LED -> J60[8]

DCX -> ~J26[12]~ J57[20]

REST -> ~J26[10]~ J57[18]

GRND -> J60[14]
